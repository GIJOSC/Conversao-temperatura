![image](https://user-images.githubusercontent.com/31870513/160693387-8cf24bf1-ded4-4f23-8a0f-1bde59c0bda6.png)




## Description
Temperature conversion it's a project developed during Iniciativa Kubernetes.
The main functionality is to convert the temperature from Celsius to Fahrenheit or from Fahrenheit to Celsius.
Using Docker Container.

:white_check_mark: - Node.js

## Quick Start

```
   
  git clone https://github.com/KubeDev/conversao-temperatura/tree/main/src
  
  cd conversao-temperatura
  
  cd src
  
  yarn start
  
```
